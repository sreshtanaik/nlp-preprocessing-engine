# NLP Preprocessing Engine

## 📌 Overview
This project implements a robust NLP preprocessing pipeline to clean and transform noisy real-world text data into meaningful tokens.

## 🚀 Features
- Lowercasing
- URL & email removal
- Number removal
- Repeated character normalization
- Token filtering
- Noise removal

## 🧪 Test Cases
Includes diverse inputs:
- Emojis
- URLs
- Slang
- Repeated characters
- Mixed case

## 📊 Analysis
- Token statistics
- Frequency analysis

## ⚙️ How to Run
1. Open the notebook (.ipynb)
2. Run all cells sequentially
